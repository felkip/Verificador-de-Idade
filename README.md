# Verificador de Idade

Este projeto é um verificador de idade em HTML, CSS e JavaScript.

## O que faz
- Solicita o ano de nascimento do usuário.
- Permite escolher entre masculino e feminino.
- Calcula a idade e exibe uma imagem correspondente à faixa etária.

## Arquivos
- index.html: formulário e estrutura da página.
- style.css: estilização do layout.
- script.js: lógica para calcular a idade e mostrar o resultado.
- assets/: pasta com as imagens usadas no projeto.

## Como usar
1. Abra o arquivo index.html em um navegador.
2. Informe o ano de nascimento.
3. Selecione o sexo e clique em "Verificar".
4. Veja o resultado e a imagem correspondente.
